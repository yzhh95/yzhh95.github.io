github+hexo搭建个人博客

发表于 2020-05-03 分类于 前端 阅读次数：
记录一下利用github和hexo搭建个人blog的过程，配合next主题的应用

1. 安装工具

换源加速下载hexo

2. hexo初始化

新建完成后，指定文件夹目录下有：

node_modules: 依赖包
public：存放生成的页面
scaffolds：生成文章的一些模板
source：用来存放你的文章
themes：主题
_config.yml: 博客的配置文件

再次输入命令：

sudo hexo g #生成静态页面
sudo hexo s #启动服务器

注：这时候就可以用浏览器打开网址： http://localhost:4000/ 来进行预览了。

3.github设置

创建仓库（注意仓库名一定写成username.github.io），设置打开pages

4. 将hexo部署到github上

先安装deploy-git ，也就是部署的命令,这样你才能用命令部署到GitHub。
deploy时要你输入username和password

再打开站点配置文件_config.yml，修改deloy

注意要用git的格式，不要用https的格式，这样之后更新不用每次都输入密码账号

至此，github的博客就建立好了，可以访问http://yzhh95.github.io查看
